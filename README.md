## Project scope

- a 2D game where the player tries to cover a barren grid-based island with plants as fast as possible
- different seeds/plants fill tiles differently (like tetris shapes)
- the player's mechanics are move and plant
- 10 levels (with different islands) of increasing difficulty, each with a level timer, aim to get the best time
- elements of level tiles can be: earth (plantable), rock (unplantable), water (unplantable), plants (planted). levels are pregenerated
- planting is instantaneous, from one click: seeds go in, tile gets watered, and then turns itself and predefined shape/area green with plants
- superfluous animations to watch the plants grow and evolve once planted if you want to chill between levels

### Simplest working version: 

- player moves and can plant has only one type of plant to fill up a simple island grid
